# Zan Services – React + Vite + Tailwind CSS

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Pages & Routes

| Route | Page |
|-------|------|
| `/` | Home |
| `/services` | Services Landing |
| `/services/web-development` | Web Development |
| `/about` | About Us |
| `/portfolio` | Portfolio |
| `/contact` | Contact |
| `/legal` | Legal (Privacy, Terms, Refund, Cookie, Disclaimer) |

## Tech Stack
- Vite + React 18
- Tailwind CSS v3
- React Router DOM v6

## Colors (tailwind.config.js)
- Primary: `orange-600` (#e84a00)
- Dark BG: `dark-900` (#130c02)
- Accent: `orange-500` (#f96a1b)

## Fonts
- Display: Syne (headings)
- Body: DM Sans
